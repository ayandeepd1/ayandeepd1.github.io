\item \textbf{Voice Integrated Home Automation} {\href{https://github.com/ayandeepd1/iot_esp_automation}{\ExternalLink} \\

-Voice recording using interrupt functionality.

}

\item \textbf{Gesture Recognition Glove} {\href{https://drive.google.com/file/d/1x3BAbsTNXpl_hvw0puoTsikSn7FdMiW4/view}{\ExternalLink}\\
-A glove like device to recognize hand gestures using AI.
-Play preassigned voice feedback to known gestures.
-Based on stm32f103 and esp32c3pico.
-Uses ADC and IMU for movements of fingers and hands.
-BLE connectivity with android app.
-FreeRtos scheduling is used to manage tasks.
-Battery powered device.
}

\item \textbf{Displaying BMP image on NeoPixel Matrices} {\href{https://github.com/ayandeepd1/image_mapping_on_neopixel_matrix}{\ExternalLink}\\
-Three 30x30 Neopixel led panels in snakelines configuration.
-Generated a series of bmp images using JINX software.
-Converted bmp image files into 2D array of RGB pixels.
-Split pixel array and stored as binary files on SD card.
-Esp32 reads SD card and display image in each Neopixel panel. }
